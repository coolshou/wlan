# wlan
wlan is the WindowsSDK sample for the WirelessAPI.


# require 
  vc2017
  
# feature
The following commands are available. Use "help xyz" to show the description of command xyz.
```
        EnumInterface(ei)
        GetDeviceList(gd)
        GetInterfaceList(gi)
        GetInterfaceCapability(gic)
        QueryInterface(qi)
        SetRadioState(srs)
        GetDriverStatistics(gds)
        Scan(scan)
        GetBssList(gbs)
        GetVisibleNetworkList(gvl)
        CreateProfile(cp)
        SetProfile(sp)
        SaveTempProfile(stp)
        GetProfile(gp)
        DeleteProfile(dp)
        DeleteProfileList(dpl)
        SetProfileList(spl)
        GetProfileList(gpl)
        Connect(conn)
        Disconnect(dc)
        Discover(disc)
        State(st)
        GetRSSI(rssi)
        GetRegkeyValue(gr)
        SetRegkeyValue(sr)
        RegisterNotif(r)
        version(ver)
        help(?)
```
